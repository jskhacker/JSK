BlackMsf

Credential Security Testing Framework

Python Ruby Metasploit API Bash Linux
◈ ABOUT BLACKMSF
⚠
Authorized Use Only. BlackMsf is strictly for use on systems you own or have explicit written permission to test. Unauthorized use is illegal and unethical. The author bears no responsibility for misuse.

BlackMsf is a purpose-built credential security testing framework designed for authorized penetration testers, security researchers, and CTF competitors. It wraps the Metasploit Framework with a sleek custom terminal UI and automates credential-strength auditing workflows.

At its core, BlackMsf is a brute-force password auditing engine — it stress-tests authentication endpoints to identify weak, default, or easily guessable credentials. This is an essential step in any authorized security audit to expose vulnerabilities before real attackers do.

Rather than running individual Metasploit commands manually, BlackMsf chains them into automated pipelines — reducing the time taken to test credential security across multiple services in a controlled lab or authorized engagement.

⚡ KEY FEATURES
  • ▸ Custom terminal UI built over the Metasploit Framework RPC API for a cleaner workflow.
  • ▸ Automated credential auditing across SSH, FTP, HTTP Basic Auth, SMB, and more.
  • ▸ Supports custom wordlists and rule-based mutation for realistic credential testing scenarios.
  • ▸ Multi-threaded engine for high-speed testing in authorized lab environments.
  • ▸ Session management dashboard — track active sessions and module results in one view.
  • ▸ Auto-generated test reports in JSON and HTML for professional documentation.
  • ▸ Modular design — easily add or remove testing modules as needed.
  • ▸ Verbose logging with timestamps for complete audit trails.
⚙ TECH STACK
Python 3
Core engine, CLI interface, threading, and report generation.
Ruby / MSFRPC
Metasploit RPC bridge for module execution and session control.
Metasploit API
Exploit and auxiliary module access for credential testing workflows.
Bash Scripts
Environment setup, dependency checks, and launcher automation.
Python Threading
Concurrent credential testing for faster authorized audits.
JSON / HTML
Structured output for audit logs and professional pentest reports.
